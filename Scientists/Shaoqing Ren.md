# Description

任少卿，中国科技大学与微软亚洲研究院联合培养博士，曾任Momenta公司研发总监。现任蔚来北美自动驾驶助理副总裁。在任少卿的职业生涯中，曾获得ImageNet 2015和MS COCO 2015挑战赛冠军，以及CVPR 2016较佳论文等荣誉。
