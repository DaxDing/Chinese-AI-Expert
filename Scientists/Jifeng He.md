# Description

何积丰院士长期从事计算机软件理论及应用研究，是程序统一理论的创立者、数据精化完备理论的奠基者，、可信软件设计理论与技术的开拓者。何积丰院士在世界上首次提出“可信人工智能”，自此世界各地开始广泛关注人工智能的安全可信问题。
